# Code-X

A real-time code editor IDE

## How to use
### 1\. Clone this Repository

```bash
git clone https://github.com/alvin-dotcom/Code-X--Collaborative-IDE
```
### 2\. Navigate to the directory

```
cd Code-X--Collaborative-IDE
```
### 3\. Install dependencies
```
npm install
```
### 5\. Run Dev Server

```
npm start
```
